## Installation

### Clone the project
```sh
git clone https://github.com/SivertGullbergHansen/Sivert.io.git portfolio
cd portfolio
```

### Install dependencies
```sh
npm i
```
or
```sh
yarn
```

### Run development server
```sh
npm run dev
```
or
```sh
yarn dev
```
